<h2> Efficient Pathfinding in Large Transportation Networks </h2>

<p>
A C++ program that compares Dijkstra's and A* algorithms on large road network graphs, showing which finds the shortest path faster.
  
To run:
make<br>
./pathfinder<br>
Then follow the menu:<br>

Load a dataset (generate synthetic if you don't have one)<br>
Select start and destination nodes<br>
Compare both algorithms to see the speed difference<br>
</p>
